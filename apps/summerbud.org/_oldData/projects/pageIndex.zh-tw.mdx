# 專案

我喜歡撰寫程式，更喜歡用程式解決人們日常會遇到的問題。

這些專案的出發點大多是我個人生活中時常遇到的問題，例如 totuslink 為質問為什麼當前社會使用如此之多「條列式」的結構，多到我們並沒有機會去思考這種結構帶來的問題；hologram 為了解決處理參考資料的痛點；Pnyx 則是為了解決當前討論的無效率。

這些專案各自表彰了我每段時間的思考，如果你對任何專案有興趣了解更深的話，我皆有附上 github repo 連結，你甚至可以寄信給我，我喜歡跟人討論這些專案的事。
